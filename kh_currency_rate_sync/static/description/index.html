Currency Rate Update Sync

Automate your daily currency exchange rates using reliable providers like ECB and Frankfurter. Supports backdated rate pulling and seamless multi-company configuration.

Overview & Key Features

  • **Reliable Providers:** Utilizes the **European Central Bank (ECB)** as the primary rate provider. If the primary fails, it seamlessly falls back to the **Frankfurter** API to ensure no interruption in rate syncing.
  • **Automated Daily Sync:** Cron jobs run daily to fetch the latest rates without manual intervention.
  • **Multi-Company Support:** Highly compatible with multi-company environments. You can configure different active currencies and synchronization settings independently for each company.
  • **Historical Rate Pulling:** Easily fetch rates for previous dates across a specified date range.
  • **Invoice Auto-Sync:** Ensure your invoices always use the latest rates by automatically triggering a sync when opening a new invoice.

Configuration & Real-Time Syncing

Configure the currencies you want to synchronize directly from your accounting settings. The interface gives you a clear overview of the sync status.

  • **Sync Currency Rate:** Define specifically which currency rates you want to sync, so you don't pollute your system with unnecessary data.
  • **Invoice Open Auto Sync:** Enable this boolean setting to automatically check and sync today's currency rate every time a user opens a new invoice. This guarantees the invoice amount is converted using the freshest available rate.
  • **Status & Last Updated:** Instantly view whether the last sync was successful and exactly when it was updated by the provider.
  • **Sync Now:** Click this button to perform an immediate manual sync for the current date.
  • **Sync Backdates:** Access a powerful tool to retrieve rates from the past.

Backdated Currency Rate Pulling

Need to fill in missing rates from last month? Click the **Sync Backdates** button to open the historical rate wizard.

  • **Start & End Date:** Define the exact period you want to fetch historical currency rates for.
  • **Specific Currency:** Select a particular currency from the dropdown if you only need rates for one. Leave it empty to automatically sync all active and configured currencies for your company within that period.

Invoice Date-Based Currency Rates

Automatically apply the precise currency rate based on the selected invoice date. When users change the invoice date, the system instantly fetches and applies the historical exchange rate for that exact day.

  • **Date-Based Precision:** Choose any past invoice date, and the system automatically retrieves and uses the official exchange rate recorded for that day.
  • **Real-Time Recalculation:** Invoice amounts and lines are instantly recalculated using the correct historical rate, requiring no manual intervention.
  • **Accurate Accounting:** Ensures your ledger remains historically accurate and compliant by applying the exact conversion rates present at the time of the transaction.
